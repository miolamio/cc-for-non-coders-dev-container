Собери еженедельный отчёт для руководства на основе квартальных данных.

Источники данных:
1. Файл ../quarterly-presentation/quarterly-results.csv — выручка, расходы, клиенты, сделки по отделам за последний квартал
2. Файл ../meeting-protocol/meeting-transcript.md — расшифровка последнего совещания (решения, задачи, дедлайны)

Формат отчёта:
- Заголовок: "Еженедельный отчёт [дата начала] — [дата конца]"
- Раздел "Финансовые итоги": общая выручка и расходы по отделам, динамика за последний месяц из CSV
- Раздел "Ключевые решения": извлеки принятые решения из расшифровки совещания
- Раздел "Задачи в работе": кто, что, до какого числа — из расшифровки
- Раздел "Проблемы и риски": отделы, где расходы превышают выручку; задачи без назначенных дедлайнов

Сохрани результат в файл reports/weekly-report-[дата].md

Тон: деловой, без воды. Только факты и цифры.
