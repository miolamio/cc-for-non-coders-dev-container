# Демо: Лендинг по описанию услуги

## Что делаем

Claude Code создаёт HTML-страницу с формой заявки, адаптивной вёрсткой и таймером обратного отсчёта из текстового описания услуги. Рабочий лендинг без дизайнера и верстальщика за пару минут.

## Время выполнения

- Claude (Opus 4.6): 2-3 минуты
- GLM 4.7: 3-5 минут (иногда адаптивность требует правки)

## Подготовка

Файл `service-brief.md` содержит описание вымышленной компании NeuroBridge -- AI-консалтинг для ритейла. Что за услуга, три пакета с ценами, целевая аудитория, команда, три отзыва клиентов, контакты. Цифры реалистичные: 23% экономии, 12% рост среднего чека.

## Что вводим в Claude Code

```
Прочитай описание услуги в service-brief.md и создай одностраничный лендинг в HTML.

Требования:
- Hero-секция с заголовком и кратким описанием
- Блок "Пакеты услуг" — три карточки с ценами
- Блок "Для кого" — целевая аудитория
- Блок "Отзывы клиентов" — карусель или карточки
- Блок "Команда" — кратко
- Форма заявки (имя, телефон, email, компания, кнопка "Оставить заявку")
- Таймер обратного отсчёта до конца спецпредложения (поставь дату через 7 дней от текущей)
- Адаптивная вёрстка — должно нормально выглядеть на телефоне
- Всё в одном HTML-файле, CSS и JS встроенные
- Дизайн: современный, тёмная тема, акцентный цвет — синий
```

## Что происходит

1. Claude читает бриф и планирует структуру страницы
2. Генерирует HTML-файл с встроенными стилями и скриптами
3. Может запустить субагент для отдельных задач (вёрстка + JS-таймер)
4. Результат: один HTML-файл, который открывается в браузере

Форма не отправляет данные (некуда), но визуально работает -- есть валидация полей. Таймер обновляется каждую секунду. Адаптивность видна через DevTools в Chrome (переключение на мобильный вид).

## Пример результата

HTML-файл (landing.html или index.html), 15-30 КБ. Содержит:
- 6-7 секций в тёмной теме с синими акцентами
- Три карточки с пакетами: Экспресс-аудит (800 000 руб.), Внедрение (от 3 000 000 руб.), Сопровождение (200 000 руб./мес.)
- Работающий таймер с днями, часами, минутами, секундами
- Форму с базовой валидацией (пустые поля подсвечиваются)
- Отзывы трёх клиентов с именами и должностями из брифа

## Что может пойти не так

**Страница выглядит слишком просто.** Claude иногда делает минималистичный дизайн. Добавьте: "Используй плавные анимации при скролле, тени у карточек, градиенты в hero-секции."

**Таймер показывает NaN.** Бывает, если Claude неправильно вычислил дату. Попросите: "Проверь, что таймер работает. Дата окончания -- ровно через 7 дней от момента открытия страницы."

**Шрифты не грузятся.** Если Claude подключил Google Fonts, а интернета нет, текст будет в стандартном шрифте. Не критично, но можно предупредить.

**Форма ничего не делает при нажатии.** Это нормально, бэкенда нет. Объясните студентам: для реальной работы формы нужен сервер. На демо достаточно визуальной работоспособности.
