# Еженедельный отчет

## Что делает этот навык

Читает данные из 4 источников в текущей папке и собирает сводный отчет за неделю.

## Источники данных

1. `sales.csv` -- данные по продажам (дата, клиент, сумма, статус)
2. `expenses.csv` -- расходы (дата, категория, сумма, описание)
3. `tasks.md` -- список задач с ответственными и статусами
4. `kpi.md` -- плановые и фактические KPI

## Инструкции

1. Прочитай все 4 файла из текущей папки.

2. Раздел "Продажи за неделю":
   - Общая сумма закрытых сделок (статус "оплачено")
   - Количество новых сделок
   - Сделки в работе (статус "счет выставлен") с суммами
   - Сравни с прошлой неделей, если данные есть

3. Раздел "Расходы":
   - Общая сумма расходов
   - Разбивка по категориям (таблица)
   - Отметь если какая-то категория сильно выросла

4. Раздел "Задачи":
   - Сколько задач выполнено, сколько в работе, сколько просрочено
   - Список просроченных задач с ответственными
   - Новые задачи, которые появились на этой неделе

5. Раздел "KPI":
   - Таблица: метрика, план, факт, % выполнения
   - Отметь метрики где факт ниже 80% от плана

6. Раздел "Итого и приоритеты":
   - 3-5 пунктов: что хорошо, что требует внимания
   - Не приукрашивай -- если цифры плохие, так и напиши

## Формат вывода

Markdown-файл `weekly-report-YYYY-MM-DD.md` (дата -- понедельник текущей недели). Таблицы в формате Markdown. Суммы с разделителями тысяч (1 234 567 руб). Без пустых разделов -- если данных нет, напиши "нет данных".
