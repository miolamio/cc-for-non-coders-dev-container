# DataPulse -- платформа управления клиентскими данными

## Общее описание

DataPulse -- это CDP (Customer Data Platform). Собирает данные о клиентах из разных источников в единые профили. На основе профилей можно строить сегменты для маркетинговых кампаний.

## Технические характеристики

Платформа работает на микросервисной архитектуре (Kubernetes), данные хранятся в ClickHouse (аналитика) и PostgreSQL (профили). Для обработки потоковых данных используется Apache Kafka -- это позволяет обрабатывать до 50 000 событий в секунду без потерь.

API: REST и GraphQL. Есть SDK для Python, JavaScript, Go. Поддержка webhook-ов для реалтайм-уведомлений.

## Интеграции

Из коробки подключаются:
- CRM: Bitrix24, amoCRM, Salesforce
- Рекламные кабинеты: Яндекс.Директ, VK Реклама, myTarget, Google Ads
- Аналитика: Яндекс.Метрика, Google Analytics 4
- Email/SMS: Mindbox, SendPulse, Unisender
- Мессенджеры: Telegram Bot API, WhatsApp Business API
- Колл-трекинг: Calltouch, CoMagic
- Произвольные источники через REST API или загрузку CSV/JSON

Всего 40+ готовых коннекторов. Новые подключаются за 2-3 дня через конструктор интеграций.

## Обработка данных

Identity Resolution: платформа сопоставляет данные из разных источников по email, телефону, cookie ID, device ID и пользовательским идентификаторам. Алгоритм вероятностного матчинга дает точность 94% на тестовых данных.

Дедупликация: автоматическое обнаружение и слияние дублей. Настраиваемые правила приоритетов -- какой источник "главнее" при конфликте данных.

Обогащение: дополнение профилей данными из открытых источников (ЕГРЮЛ, 2ГИС для B2B) и партнерских DMP.

## Сегментация и аналитика

Визуальный конструктор сегментов: условия по атрибутам профиля, событиям, RFM-метрикам. Сегменты пересчитываются автоматически (настраиваемый интервал, минимум 15 минут).

Встроенные отчеты: воронки, когортный анализ, LTV, retention. Дашборды настраиваются через drag-and-drop. Экспорт в BI-системы через SQL-доступ к хранилищу.

Предиктивная аналитика: модели оттока и вероятности покупки на базе градиентного бустинга (LightGBM). Модели обучаются на данных клиента, переобучение раз в неделю.

## Безопасность и compliance

- Хранение данных в РФ (дата-центры Selectel, Москва и Санкт-Петербург)
- Соответствие 152-ФЗ о персональных данных
- Шифрование данных at rest (AES-256) и in transit (TLS 1.3)
- RBAC: ролевая модель доступа с настройкой на уровне полей
- Аудит-лог всех действий пользователей
- SLA 99.9% (подтверждено за последние 12 месяцев: 99.94%)

## Производительность

- Объем хранения: до 500 млн профилей на одном инстансе
- Скорость запросов: медианный ответ API -- 120 мс, 95-й перцентиль -- 350 мс
- Пересчет сегмента на 10 млн профилей: ~45 секунд
- Импорт данных: до 200 000 записей/сек через bulk API
