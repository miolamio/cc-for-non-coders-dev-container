# Демо: сравнение 7 предложений от подрядчиков

## Что показываем

Claude разбирает 7 коммерческих предложений на ремонт офиса в разных форматах (формальные КП, письма, табличная смета) и строит единую сравнительную матрицу.

Время на демо: разбор предложений ~2-3 минуты, с обсуждением ~10 минут.

## Основной промпт

```
В папке proposals/ лежат 7 коммерческих предложений от подрядчиков на ремонт нашего офиса (287 кв.м). Форматы разные: КП, письма, таблицы.

Разбери каждое предложение и построй сводную таблицу сравнения:
- Подрядчик
- Итоговая стоимость (приведи к общему знаменателю: с НДС и без, что включено, что нет)
- Сроки (от подписания до заезда)
- Гарантия
- Размер аванса
- Что включено, что нет (кондиционеры, пожарка, серверная, стекло)
- Форма собственности (ООО, ИП)

Отдельно напиши: какие предложения неполные (нет каких-то данных), где нужно запросить уточнения, и какие 3 подрядчика выглядят лучше всего по соотношению цена/качество/сроки.
```

## Ожидаемый результат

Claude должен выдать:

1. **Сводную таблицу** со всеми 7 подрядчиками по указанным параметрам
2. **Замечания по неполным предложениям** (МосРемонт без сроков, ЕвроСтандарт без кондиционеров)
3. **Скорректированные цены** с учетом того, что не включено (для честного сравнения)
4. **Топ-3 рекомендации** с аргументами

Примерная структура:

```
## Сводная таблица

|----------|---------------|-----------|------------|-----------|-------------|--------------|---------------|
| Цена без НДС | 5 492 000 | 5 450 000 | 5 611 000 | 3 993 730 | 4 300 000* | 7 295 000 | 3 445 000* |
| ...

* -- неполная комплектация, см. примечания

## Примечания
...

## Рекомендация
1. СтройПрогресс -- оптимальный баланс цена/сроки/гарантия...
2. ...
```

## На что обратить внимание студентов

1. **Claude работает с разными форматами.** Формальное КП, email, CSV-смета -- все в одном анализе. В жизни предложения приходят именно так, каждый подрядчик присылает по-своему.

2. **Claude замечает нюансы.** Разница НДС/без НДС, неполная комплектация, отсутствие сроков -- все это Claude должен подсветить.

3. **Скорректированные цены важнее "итого из КП".** Если подрядчик не включил кондиционеры, его низкая цена -- обманка. Claude пересчитывает для честного сравнения.

4. **Это материал для переговоров, а не финальное решение.** С этой таблицей можно идти на встречу с каждым подрядчиком и задавать конкретные вопросы.

## Уточняющий промпт

```
Теперь для каждого подрядчика составь список вопросов, которые нужно задать перед принятием решения. Например: МосРемонту -- сроки, ЕвроСтандарту -- стоимость кондиционеров, Мастер Ремонту -- почему нет стеклянных перегородок.
```

## Частые проблемы

**Claude не открывает CSV-файл корректно**
- Попросите: "Разбери файл 04-mosremont-estimate.csv как таблицу с разделителем-запятой"
- Или предварительно конвертируйте CSV в MD-таблицу

**Слишком длинная таблица**
- Попросите: "Сначала краткую таблицу (цена, сроки, гарантия), потом детали отдельно"

**Claude неправильно суммирует позиции из CSV**
- Проверьте итоговую сумму вручную. Если есть расхождение -- хороший повод обсудить ограничения LLM в арифметике

**Claude не замечает неполноту предложений**
- Подскажите: "Проверь, у всех ли подрядчиков указаны сроки, гарантия, состав работ. Кто что-то не включил?"
