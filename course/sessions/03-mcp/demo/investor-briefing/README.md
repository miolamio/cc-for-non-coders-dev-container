# Демо: досье перед встречей с инвестором

## Что показываем

Claude через Tavily MCP ищет информацию об инвестиционном фонде и собирает двухстраничный брифинг для встречи. Никаких файлов готовить не нужно -- все берется из интернета.

Время на демо: генерация брифинга ~2-3 минуты, с разбором ~10 минут.

## Подготовка

1. MCP-сервер Tavily должен быть подключен (см. демо `mcp-setup`)
2. Проверьте, что API-ключ Tavily работает: попросите Claude найти что-нибудь простое

**Тест-промпт для проверки Tavily:**
```
Найди через Tavily, кто основал компанию Automatica и чем она занимается.
```

Если ответ пришел -- Tavily работает, можно переходить к основному демо.

## Сценарий

Вы -- основатель стартапа в сфере EdTech. Через два дня встреча с представителем фонда Runa Capital. Нужно понять: во что они инвестируют, какие сделки закрывали недавно, кто принимает решения, на каком языке с ними разговаривать.

**Запасной вариант:** если Tavily находит мало информации по Runa Capital (фонд мог уйти из публичного поля или сменить стратегию), используйте фонд Almaz Capital Partners или Baring Vostok. Промпт тот же, меняется только имя фонда.

## Основной промпт

```
У меня послезавтра встреча с инвестором из Runa Capital. Мой стартап -- EdTech-платформа для корпоративного обучения.

Найди через интернет и собери для меня брифинг:

1. Общая информация о фонде: год основания, размер, где офисы
2. Инвестиционный фокус: в какие сектора и стадии вкладывают
3. Последние 5-7 сделок (название компании, раунд, сумма, дата)
4. Ключевые партнеры фонда -- кто есть кто, чем известны
5. Есть ли в портфеле EdTech-компании или что-то близкое
6. Talking points: о чем стоит говорить на встрече, зная их портфель

Формат: структурированный документ, который можно прочитать за 10 минут.
```

## Что должно произойти

1. Claude сделает несколько поисковых запросов через Tavily (обычно 3-6)
2. Соберет информацию из разных источников
3. Скомпилирует брифинг по указанной структуре
4. Добавит talking points, адаптированные под EdTech

## Пример готового брифинга

Вот приблизительно что должен выдать Claude (конкретные цифры могут отличаться в зависимости от того, что найдет Tavily):

```
# Брифинг: Runa Capital

Дата подготовки: 2 марта 2026
Цель: подготовка к встрече, EdTech-стартап (корп. обучение)

## О фонде
- Год основания: 2010
- Основатели: Дмитрий Чихачев, Сергей Белоусов, Андре Блюм
- Размер фондов: $200M+ (суммарно несколько фондов)
- Офисы: Люксембург, Хельсинки (исторически также Москва и Кремниевая долина)
- Фокус: deep tech, инфраструктурное ПО, B2B SaaS, AI/ML
- Стадии: Seed, Series A

## Последние сделки
| Компания       | Сектор        | Раунд   | Сумма   | Дата     |
|----------------|---------------|---------|---------|----------|
| Esempio AI     | AI/ML         | Seed    | $3M     | Q4 2025  |
| DataLayer      | Инфраструктура| Series A| $12M    | Q3 2025  |
| ...            | ...           | ...     | ...     | ...      |

(Claude заполнит реальными данными из поиска)

## Ключевые партнеры
- Дмитрий Чихачев -- управляющий партнер, фокус на deep tech.
  Ранее -- со-основатель Parallels (Odin)
- Андре Блюм -- партнер, фокус на B2B SaaS

## EdTech в портфеле
(Claude укажет, есть ли EdTech-компании. Если нет -- это тоже
полезная информация: значит, ваш стартап для них новый сектор.)

## Рекомендации к встрече
1. Фонд фокусируется на deep tech -- подчеркните технологическую
   составляющую платформы, а не контент
2. Им важна масштабируемость -- покажите unit-экономику
   и потенциал роста за пределами РФ
3. ...
```

## На что обратить внимание студентов

1. **Claude делает несколько запросов подряд.** Он сам решает, что искать: сначала общую информацию, потом сделки, потом конкретных людей. Это не один запрос в Google, а цепочка.

2. **Информация может быть неточной.** Суммы сделок и даты надо проверять по первоисточникам. Скажите студентам: это черновик для подготовки, а не юридический документ.

3. **Tavily vs обычный поиск.** Tavily возвращает извлеченный текст, а не ссылки. Claude получает контент, а не список URL.

4. **Формат результата зависит от промпта.** Если не указать структуру, Claude выдаст текст в свободной форме. Конкретная структура в промпте дает конкретный результат.

## Вариации для продвинутых

**Если осталось время, покажите уточняющий промпт:**

```
Теперь найди информацию о партнере фонда, который отвечает за EdTech-направление (если такой есть). Что он писал в блогах или говорил на конференциях об EdTech? Какие метрики его интересуют?
```

**Утренний дайджест (второй мини-кейс):**

```
Найди новости за последнюю неделю по теме "EdTech корпоративное обучение Россия". Отфильтруй рекламу и пресс-релизы. Оставь только реальные события: сделки, запуски продуктов, регуляторные изменения. Выдай сводку на одну страницу.
```

## Частые проблемы

**Tavily возвращает мало результатов**
- Попробуйте более общий запрос
- Информация на русском хуже индексируется -- попросите Claude искать и на английском тоже

**Информация устаревшая или фонд плохо гуглится**
- Переключитесь на запасной фонд (Almaz Capital Partners или Baring Vostok)
- Промпт менять не нужно, только имя фонда

**Claude придумывает суммы сделок**
- Это бывает. Скажите студентам, что цифры из брифинга нужно проверять
- Попросите Claude добавить ссылки на источники: "По каждой сделке укажи URL источника"
