# Идеи для внутренних инструментов

Собираю сюда всё, что хотелось бы автоматизировать.

## 1. Генератор КП
Ввёл: имя клиента, задачу, бюджет, сроки.
Получил: PDF в нашем стиле с логотипом, содержанием, условиями.
Сейчас каждое КП делаем руками в Google Docs. На одно уходит 2-3 часа.

## 2. Дашборд проектов
Сейчас всё в Notion, но нет визуального обзора: что горит, что в графике, что задерживается.
Хочется что-то типа: зелёный / жёлтый / красный по каждому проекту.

## 3. Автоматическая сводка по финансам
Еженедельно: сколько пришло, сколько ушло, какие счета не оплачены.
Сейчас раз в месяц считаем вручную с бухгалтерией.

## 4. Бот для таймтрекинга
Команда не ведёт учёт времени. Без этого не посчитать реальную себестоимость проекта.
Нужен простой бот в Telegram: нажал кнопку -- начал проект, нажал -- закончил.

## 5. Шаблон еженедельного отчёта
Каждый пишет что сделал за неделю -- в своём формате. Хочется единый формат:
- Что сделано
- Что в работе
- Что блокирует
- План на следующую неделю

Реальность: половину из этого можно сделать в n8n + Google Sheets. Генератор КП -- через Claude Code + шаблон. Надо выделить время.
