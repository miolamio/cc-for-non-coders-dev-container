# Демо: Финансовая сводка из банковской выписки

## Что здесь

Файл `bank-statement.csv` -- выписка по расчётному счёту маленького digital-агентства за январь--март 2025. Около 95 операций: зарплаты, аренда, маркетинг, налоги, оплаты от клиентов.

Разделитель -- точка с запятой (стандарт для российских банков). Суммы со знаком: минус -- расход, плюс -- поступление.

## Время

На Claude Pro/Max: ~30-60 секунд.
На GLM 4.7: ~1-2 минуты.

Claude прочитает CSV, напишет HTML с JavaScript и сохранит файл. Основное время уходит на генерацию кода для графиков.

## Цель демо

Показать, как Claude Code берёт сырой CSV и за один промпт делает из него HTML-страницу с графиками и таблицами. Без Excel, без формул, без ручной вёрстки.

## Подготовка

1. Открыть терминал в папке `financial-dashboard/`
2. Запустить `claude` (или убедиться, что Claude Code уже запущен)
3. Убедиться, что CSV на месте: `ls` должен показать `bank-statement.csv`

## Промпт для демонстрации

Вводим в Claude Code:

```
Посмотри файл bank-statement.csv -- это выписка из банка за 3 месяца. Сделай из неё HTML-дашборд:
- Общая сводка: доходы, расходы, баланс за каждый месяц
- Столбчатая диаграмма (bar chart) расходов по категориям
- Круговая диаграмма (pie chart) долей расходов
- Выручка по клиентам (кто сколько заплатил за 3 месяца)
- Линейный график (line chart) динамики: доходы и расходы помесячно
Используй Chart.js для графиков. Всё на одной странице, на русском.
```

## Что произойдёт

Claude Code:
1. Прочитает CSV, разберёт структуру (разделитель, колонки, формат дат)
2. Напишет HTML-файл с встроенным JavaScript
3. Подключит Chart.js через CDN
4. Сгенерирует таблицы, графики, сводки
5. Сохранит файл (обычно `dashboard.html`)

## Пример результата

После открытия `dashboard.html` в браузере вы увидите примерно следующее:

**Сводная таблица (верхняя часть страницы):**

| Месяц   | Доходы       | Расходы      | Баланс       |
|---------|-------------|-------------|-------------|
| Январь  | 1 950 000 ₽ | 1 120 000 ₽ | +830 000 ₽  |
| Февраль | 1 680 000 ₽ | 1 085 000 ₽ | +595 000 ₽  |
| Март    | 2 210 000 ₽ | 1 150 000 ₽ | +1 060 000 ₽|

(Суммы приблизительные -- зависят от того, как Claude сгруппирует операции.)

**Столбчатая диаграмма расходов по категориям** -- столбики по категориям: Зарплата (~432 000/мес), Аренда (185 000/мес), Налоги (~186 000/мес), Маркетинг (~80 000/мес), Софт, Офис, Банковские услуги.

**Таблица выручки по клиентам** -- строки: РетейлГрупп, ТехноСтрой, ИнноТех, ФинансАвто, БрендСервис, МаркетПлюс -- с суммой за 3 месяца по каждому.

**Линейный график динамики** -- две линии (доходы и расходы) по месяцам, видно, что март был лучшим по выручке.

## На что обратить внимание студентов

1. Claude сам разобрался с форматом. Разделитель `;`, русские названия, формат дат DD.MM.YYYY -- ничего из этого не пришлось объяснять.

2. Категоризация уже была в файле, но Claude мог бы сделать её сам. Можно упомянуть: "если у вас выписка без категорий -- Claude сам разложит по группам".

3. Не пришлось писать код, знать HTML или JavaScript. Claude сам выбрал инструменты (Chart.js), сам сверстал.

4. Файл работает автономно. Открывается в любом браузере, Chart.js подтягивается из CDN. Можно отправить коллеге -- он увидит то же самое.

## Вариации (если останется время)

Можно попросить Claude доработать:

```
Добавь таблицу топ-5 самых крупных расходных операций. И выдели красным категории, где расходы выросли по сравнению с прошлым месяцем.
```

Или показать, как улучшить формулировку задачи:

- Плохо: "Разбери выписку" (слишком размыто)
- Лучше: "Категоризируй расходы, покажи динамику за 3 месяца, выдели топ-клиентов по выручке"

Это переход к теме из "Разбор и закрепление" -- как формулировать задачи.

## Если что-то пошло не так

Графики не отображаются:
```
Графики не видно. Встрой Chart.js прямо в HTML-файл, без CDN -- у меня нет интернета.
```

Суммы считаются неправильно (например, расход и доход поменялись местами):
```
Суммы в CSV со знаком: минус -- это расход, плюс -- поступление. Формат: -185000.00, разделитель дробной части -- точка. Пересчитай.
```

Claude создал пустую страницу или не создал файл:
```
Создай HTML-файл dashboard.html в текущей папке. Открой bank-statement.csv, разбери данные и сгенерируй страницу с графиками.
```

Дашборд выглядит слишком просто (нет графиков, только текст):
```
Добавь Chart.js графики: bar chart расходов по категориям, line chart динамики по месяцам, pie chart долей расходов. Оформи с CSS -- белый фон, карточки с тенями, нормальные отступы.
```

## Возможные проблемы

| Ситуация | Что делать |
|----------|------------|
| Claude создал файл, но графики не отображаются | Проверить, есть ли интернет (нужен CDN для Chart.js). Или попросить: "Встрой Chart.js прямо в файл, без CDN" |
| Неправильно распарсил суммы | Бывает из-за точки/запятой в числах. Попросить: "Суммы в формате -185000.00, разделитель дробной части -- точка" |
| Создал слишком простой дашборд | Уточнить промпт, добавить конкретики: "добавь сортировку по клиентам, графики с подписями значений" |
| Открыл CSV, но не создаёт HTML | Явно сказать: "Создай HTML-файл dashboard.html в текущей папке" |
